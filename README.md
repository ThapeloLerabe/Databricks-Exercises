**📘 SQL Practice Portfolio — Exercises 1 to 4**
A consolidated collection of foundational SQL work, covering queries, filtering, joins, grouping, and CASE statements.

🧾 About This Repository
This repository contains my completed SQL practice exercises (Exercises 1–4), each focusing on a different set of core SQL concepts. The goal of this project is to strengthen my skills as a junior data professional by working through realistic query scenarios including data retrieval, filtering, joins, aggregation, and conditional logic.
All solutions include both the SQL queries and the expected output tables.
